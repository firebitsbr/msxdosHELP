## 4. How to use

Syntax:

HELP [?][LIST|L][<command>]
  
* __?__         About the application
* __LIST__      Displays the list of commands with a short description
* __L__         The same as LIST 
* __<command>__ It shows information about the use of a MSXDOS command

Examples:

```  
 A>HELP LIST
```

```
 A>HELP DIR
```
